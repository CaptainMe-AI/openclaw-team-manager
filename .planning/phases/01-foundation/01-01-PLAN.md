    **Step 2: Overwrite config/database.yml** (per D-04):
    Replace the generated database.yml with:
    ```yaml
    default: &default
      adapter: postgresql
      encoding: unicode
      pool: <%= ENV.fetch("RAILS_MAX_THREADS") { 5 } %>
      host: <%= ENV.fetch("DATABASE_HOST", "localhost") %>
      port: <%= ENV.fetch("DATABASE_PORT", "5411") %>

      <<: *default
      database: openclaw_team_manager_development

    test:
      <<: *default
      database: openclaw_team_manager_test

    production:
      <<: *default
      database: openclaw_team_manager_production
    ```

    **IMPORTANT (Pitfall 10):** Rails 8 may generate separate database entries for `queue`, `cache`, and `cable` under each environment. Remove ALL of those extra database entries. Keep only the single database per environment as shown above. Solid Cable will share the primary database -- we will handle its migration in the primary database.

    **Step 3: Configure structure.sql format** (per D-06):
    In `source/dashboard/config/application.rb`, inside the `class Application < Rails::Application` block, add:
    ```ruby
    config.active_record.schema_format = :sql
    ```

    **Step 4: Configure UUID primary keys globally** (per D-03):
    Create `source/dashboard/config/initializers/generators.rb`:
    ```ruby
    # frozen_string_literal: true

    Rails.application.config.generators do |g|
      g.orm :active_record, primary_key_type: :uuid
    end
    ```

    **Step 5: Create pgcrypto migration** (per Pitfall 4):
    ```bash
    cd source/dashboard
    bin/rails generate migration EnablePgcrypto
    ```
    Edit the generated migration file to contain:
    ```ruby
    # frozen_string_literal: true

    class EnablePgcrypto < ActiveRecord::Migration[8.0]
      def change
        enable_extension "pgcrypto" unless extension_enabled?("pgcrypto")
      end
    end
    ```
